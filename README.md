# Gestiona Asesoría - Landing Page

Landing page para **Gestiona Asesoría**, especialistas en asesoría fiscal, contable y laboral para el sector agroalimentario en Huelva (especialmente fresa y arándano). El diseño está enfocado en destacar cómo los servicios se adaptan al ciclo real de cultivo (inversión y recolección).

## Tecnologías Utilizadas

- **HTML5 & CSS3**: Maquetación y estilos modernos (Glassmorphism, custom properties, Flexbox/Grid).
- **JavaScript (ES6+)**: Lógica de interactividad y manipulación del DOM.
- **Vite**: Herramienta de compilación y servidor de desarrollo ultrarrápido.
- **GSAP (GreenSock)** y **ScrollTrigger**: Animaciones complejas y efectos al hacer scroll.
- **Lucide Icons**: Sistema de iconos SVG.

## Requisitos Previos

Se requiere tener instalado [Node.js](https://nodejs.org/) (versión 16 o superior).

## Instalación y Uso

Sigue estos pasos para levantar el proyecto en tu entorno local:

1. Abre una terminal en la raíz del proyecto.
2. Instala las dependencias necesarias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo en local:
   ```bash
   npm run dev
   ```
4. Abre la URL proporcionada en tu navegador (por defecto suele ser `http://localhost:5173/`).

## Scripts Disponibles

En el directorio del proyecto, puedes ejecutar los siguientes comandos:

- `npm run dev`: Ejecuta la aplicación en modo de desarrollo.
- `npm run build`: Compila y minifica el proyecto para producción en la carpeta `dist/`.
- `npm run preview`: Arranca un servidor web local para visualizar el proyecto compilado que se encuentra en `dist/`.

## Estructura Principal del Proyecto

- `index.html`: Archivo HTML principal de la landing page.
- `style.css`: Hoja de estilos con variables, utilidades y componentes.
- `main.js`: Script principal donde se inicializan las animaciones y la interactividad.
- `public/`: Carpeta para recursos estáticos (imágenes, vídeos, logos).
- `package.json` y `vite.config.js`: Archivos de configuración de dependencias y de la herramienta Vite.
